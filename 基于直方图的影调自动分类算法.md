\# 基于直方图的影调自动分类算法



\### 1. 概述

本算法将摄影中的十大影调分类转化为可计算的数学模型。分类基于两个维度：

\- \*\*基调\*\*：由直方图主峰值的位置决定，分为高调、中调、低调；

\- \*\*跨度\*\*：由像素亮度分布的范围决定，分为长调、中调、短调；

\- 外加一种特殊的 \*\*全长调\*\*（U 形直方图）。



算法通过分析灰度直方图的波峰、分位数和分区占比，输出明确的影调类别及置信度，后者反映分类结果的可靠程度。



\### 2. 输入与预处理

\- \*\*输入\*\*：任意 RGB 彩色图像或灰度图像。

\- \*\*灰度转换\*\*：使用 Rec. 709 标准 + sRGB Gamma 校正，转换为 0–255 的灰度图。  

&#x20; （先将 sRGB 转线性光，用 0.2126 R + 0.7152 G + 0.0722 B 计算亮度，再压回 sRGB gamma。）

\- \*\*直方图\*\*：计算 256 级灰度直方图 `hist\[i]`，并归一化使总和为 1。

\- \*\*平滑处理（必需）\*\*：对 `hist` 进行一维高斯平滑（σ=1.2）或均值滤波（窗口宽度5），得到 `h\_smooth`，用于提取稳定波峰和计算尖锐度。



\### 3. 核心统计量定义

基于预处理后的直方图，提取以下关键量：



| 统计量 | 符号 | 计算方法 | 用途 |

| :--- | :--- | :--- | :--- |

| 平滑后直方图峰值位置 | `peak\_bin` | `argmax(h\_smooth)` | 判断基调 |

| 波峰尖锐度 | `sharpness` | 峰值 / 邻域均值（见 8.2 节） | 基调置信度 |

| 第10百分位数 | `P10` | 累积分布首次达到 10% 的亮度值 | 判断跨度下限 |

| 第90百分位数 | `P90` | 累积分布首次达到 90% 的亮度值 | 判断跨度上限 |

| 暗部占比 | `low\_ratio` | `sum(hist\[0:86])` | 全长调检测 |

| 亮部占比 | `high\_ratio` | `sum(hist\[171:256])` | 全长调检测 |

| 中间调占比 | `mid\_ratio` | `sum(hist\[86:171])` | 全长调检测 |



> \*\*区间定义\*\*  

> - 暗部（Shadows）：0–85  

> - 中间调（Midtones）：86–170  

> - 亮部（Highlights）：171–255



\### 4. 分类决策流程

分类按优先级顺序执行：先排查全长调，再判定基调，最后判定跨度。



\#### 4.1 步骤一：全长调检测

\*\*判定条件（需同时满足）\*\*：

\- `low\_ratio  > 0.12`

\- `high\_ratio > 0.12`

\- `mid\_ratio  < 0.35`



若全部满足，则直接判定为 \*\*全长调\*\*，结束分类。  

若两端很高但中间不空，会被后续基调+跨度正确归类（如低长调或高长调）。



\#### 4.2 步骤二：基调判定（基于波峰位置）

\- 若 `peak\_bin ≥ 171` → \*\*高调\*\*

\- 若 `peak\_bin ≤ 85` → \*\*低调\*\*

\- 否则       → \*\*中调\*\*



\#### 4.3 步骤三：跨度判定（基于分位数）

计算 `P10` 和 `P90`：

\- \*\*长调\*\*：`P10 ≤ 20` \*\*且\*\* `P90 ≥ 235`

\- \*\*短调\*\*：`(P90 - P10) < 80`

\- \*\*中调\*\*：不满足上述两者



\#### 4.4 组合输出

将基调结果与跨度结果拼接，得到最终影调名称：

> 高长调、高中调、高短调、中长调、中中调、中短调、低长调、低中调、低短调、全长调



\### 5. 阈值参数与调优建议

下表列出所有可调参数及其默认值，可根据实际图像集微调。



| 参数 | 默认值 | 说明 |

| :--- | :--- | :--- |

| 暗部区间 | \[0, 85] | 与调性阈值一致 |

| 亮部区间 | \[171, 255] | 与调性阈值一致 |

| 基调高调阈值 | `peak\_bin ≥ 171` | 峰值位于亮部即为高调 |

| 基调低调阈值 | `peak\_bin ≤ 85` | 峰值位于暗部即为低调 |

| 全长调暗/亮占比下限 | 0.12（12%） | 两端至少拥有这些像素才构成U形 |

| 全长调中间占比上限 | 0.35（35%） | 中间区域不能太拥挤 |

| 长调 P10 阈值 | 20 | 必须有10%的像素≤此亮度 |

| 长调 P90 阈值 | 235 | 必须有10%的像素≥此亮度 |

| 短调跨度阈值 | 80 | 分位数跨度小于此值即为短调 |

| 波峰尖锐度窗口 | 10 | 计算邻域均值的单侧范围 |

| 置信度边界缓冲 | 15 | 调性/跨度靠近边界时的置信度过渡宽度 |



\### 6. 伪代码实现（核心分类）

```python

def classify\_tonality(image):

&#x20;   # 1. 灰度化与直方图

&#x20;   gray = rgb\_to\_gray(image)

&#x20;   hist, \_ = np.histogram(gray, bins=256, range=(0, 256))

&#x20;   hist = hist / hist.sum()

&#x20;   h\_smooth = gaussian\_filter1d(hist, sigma=1.2)



&#x20;   # 2. 分区占比

&#x20;   low\_ratio  = hist\[:86].sum()

&#x20;   mid\_ratio  = hist\[86:171].sum()

&#x20;   high\_ratio = hist\[171:].sum()



&#x20;   # 3. 全长调优先

&#x20;   if low\_ratio > 0.12 and high\_ratio > 0.12 and mid\_ratio < 0.35:

&#x20;       return "全长调", full\_confidence(low\_ratio, mid\_ratio, high\_ratio, gray.min(), gray.max())



&#x20;   # 4. 基调

&#x20;   peak = np.argmax(h\_smooth)

&#x20;   if peak >= 171:

&#x20;       key = "高"

&#x20;   elif peak <= 85:

&#x20;       key = "低"

&#x20;   else:

&#x20;       key = "中"



&#x20;   # 5. 跨度

&#x20;   cdf = np.cumsum(hist)

&#x20;   P10 = np.searchsorted(cdf, 0.10)

&#x20;   P90 = np.searchsorted(cdf, 0.90)



&#x20;   if P10 <= 20 and P90 >= 235:

&#x20;       span = "长调"

&#x20;   elif (P90 - P10) < 80:

&#x20;       span = "短调"

&#x20;   else:

&#x20;       span = "中调"



&#x20;   return key + span, (key\_conf(key, peak, h\_smooth), span\_conf(span, P10, P90))

```



\### 7. 边界情况与鲁棒性

\- \*\*纯色图像\*\*：`P10 ≈ P90`，跨度接近 0，波峰等于该亮度值，会被正确归为对应基调的短调。

\- \*\*噪声像素\*\*：极少量死黑/死白像素不会推动分位数 P10/P90，分类稳健；波峰平滑进一步降低伪峰影响。

\- \*\*全长调与极暗/极亮长调的区分\*\*：通过中间调上限 0.35 有效隔离。若仅两端高但中间不空，则归入低长调或高长调。

\- \*\*多峰直方图\*\*：取平滑后全局最高峰，能正确反映画面主体亮度倾向。



\### 8. 置信度估计

每个分类结果均附带一个 0.5（不确定）到 1.0（非常确定）的置信度值，由波峰形态、边界距离和分布特征综合计算。



\#### 8.1 全长调置信度

由两端强度、中间空洞度和范围覆盖三方面决定：

\- \*\*两端因子\*\*：`edge = min(low\_ratio, high\_ratio) / 0.15`（截断至 \[0,1]）

\- \*\*中空因子\*\*：`hollow = 1.0 - mid\_ratio / 0.35`（截断至 \[0,1]）

\- \*\*范围因子\*\*：`range\_full = (max\_val - min\_val) / 240`（截断至 \[0,1]）

\- 合成：`confidence = 0.5 + 0.5 × (0.4·edge + 0.4·hollow + 0.2·range\_full)`



\#### 8.2 基调置信度

基于\*\*波峰尖锐度\*\*和\*\*峰值与边界距离\*\*：

\- \*\*波峰尖锐度\*\*：  

&#x20; 取平滑直方图峰值 `peak\_val`，以窗口 `w=10` 计算邻域均值 `neighbor\_mean`（排除峰值自身），  

&#x20; `sharpness = clamp(peak\_val / neighbor\_mean, 1.0, 5.0)`  

&#x20; 将 sharpness 线性映射到因子 `peak\_factor = 0.7 + 0.3 × (sharpness - 1) / 4`（范围 0.7\~1.0）。

\- \*\*边界距离因子\*\*：  

&#x20; 设 `buffer=15`。  

&#x20; - 高调：`dist = peak\_bin - 171`  

&#x20; - 低调：`dist = 85 - peak\_bin`  

&#x20; - 中调：`dist = min(peak\_bin - 85, 171 - peak\_bin)`  

&#x20; `dist\_factor = 0.5 + 0.5 × min(max(0, dist) / buffer, 1.0)`

\- 基调置信度：`key\_confidence = max(0.5, peak\_factor × dist\_factor)`



\#### 8.3 跨度置信度

\- \*\*长调\*\*（`P10 ≤ 20` 且 `P90 ≥ 235`）：  

&#x20; `low\_margin = min((20 - P10) / 20, 1.0)`（若 P10 更低则取 1）  

&#x20; `high\_margin = min((P90 - 235) / 20, 1.0)`  

&#x20; `span\_margin = min(low\_margin, high\_margin)`  

&#x20; `confidence = 0.5 + 0.5 × span\_margin`

\- \*\*短调\*\*（`span < 80`）：  

&#x20; `compactness = 1.0 - (P90 - P10) / 80`  

&#x20; `confidence = 0.5 + 0.5 × max(0, compactness)`

\- \*\*中调\*\*：  

&#x20; `dist\_to\_long = max(0, (P10 - 20) / 20, (235 - P90) / 20)`  

&#x20; `dist\_to\_short = max(0, (span - 80) / 80)`  

&#x20; `confidence = 0.5 + 0.5 × max(dist\_to\_long, dist\_to\_short, 0.2)`



所有置信度最终截断至 \[0.5, 1.0]。



\#### 8.4 置信度伪代码

```python

def full\_confidence(low, mid, high, min\_v, max\_v):

&#x20;   edge = min(min(low, high) / 0.15, 1.0)

&#x20;   hollow = max(0, 1.0 - mid / 0.35)

&#x20;   range\_f = min((max\_v - min\_v) / 240, 1.0)

&#x20;   return 0.5 + 0.5 \* (0.4\*edge + 0.4\*hollow + 0.2\*range\_f)



def key\_confidence(key, peak, h\_smooth):

&#x20;   # 尖锐度

&#x20;   w = 10

&#x20;   neighbors = np.concat(\[h\_smooth\[max(0,peak-w):peak], h\_smooth\[peak+1:peak+w+1]])

&#x20;   nm = np.mean(neighbors) if len(neighbors) > 0 else 1e-6

&#x20;   sharpness = min(5.0, max(1.0, h\_smooth\[peak] / nm))

&#x20;   peak\_factor = 0.7 + 0.3 \* (sharpness - 1) / 4.0

&#x20;   # 距离

&#x20;   buffer = 15

&#x20;   if key == '高':

&#x20;       dist = peak - 171

&#x20;   elif key == '低':

&#x20;       dist = 85 - peak

&#x20;   else:

&#x20;       dist = min(peak - 85, 171 - peak)

&#x20;   dist\_factor = 0.5 + 0.5 \* min(max(0, dist) / buffer, 1.0)

&#x20;   return max(0.5, peak\_factor \* dist\_factor)



def span\_confidence(span\_type, P10, P90):

&#x20;   span = P90 - P10

&#x20;   if span\_type == '长调':

&#x20;       low\_m = min((20 - P10) / 20, 1.0) if P10 <= 20 else 0

&#x20;       high\_m = min((P90 - 235) / 20, 1.0) if P90 >= 235 else 0

&#x20;       return 0.5 + 0.5 \* min(low\_m, high\_m)

&#x20;   elif span\_type == '短调':

&#x20;       return 0.5 + 0.5 \* max(0, 1.0 - span/80)

&#x20;   else:  # 中调

&#x20;       d\_long = max(0, (P10 - 20)/20, (235 - P90)/20)

&#x20;       d\_short = max(0, (span - 80)/80)

&#x20;       return 0.5 + 0.5 \* max(d\_long, d\_short, 0.2)

```



\### 9. 局限性与拓展

\- 算法基于灰度直方图，忽略色彩分布。若色彩强烈影响影调感受，可转至 Lab 空间仅使用 L 通道。

\- 阈值均为经验固定值，特殊艺术风格（如极高反差）可能需要调整。

\- 输出为离散十类，实际影调具有连续性。可输出“基调倾向值”与“跨度程度值”作为连续量。

\- 置信度权重可在有标注的数据集上进一步校准。
